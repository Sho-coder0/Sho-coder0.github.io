CC5. Accessing data: Scraper and API

This chart uses data retrieved from the World Bank API, which allows direct access to cross-country macroeconomic indicators.

The base url of the API is:

  • https://api.worldbank.org/v2/

The following parameters are used in the request:

  • country: ISO2 country codes (CA, FR, DE, IT, JP, GB, US)
  • indicator: Unemployment rate
  • date: time range from 2015 to 2024
  • format: JSON response format
  • per_page: returns all observations in a single response

The final API call used in this project is:
https://api.worldbank.org/v2/country/CA;FR;DE;IT;JP;GB;US/indicator/SL.TLF.TOTL.IN?date=2015:2024&format=json&per_page=20000
